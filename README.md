# 📊 Sales Dashboard - 2026

A professional **Sales Dashboard** built in **Microsoft Excel** to analyze and monitor sales performance across different regions, products, and salespersons. The dashboard provides interactive visualizations using Pivot Tables, Pivot Charts, Slicers, and KPI Cards, enabling users to make data-driven business decisions quickly.

# 📈 Dashboard Visualizations

### 1. Sales by Product
Displays the total sales generated by each product category using a horizontal bar chart.

### 2. Sales by Region
Shows the percentage contribution of each region using a doughnut chart.

### 3. Sales by Salesperson
Compares the performance of individual salespersons using a column chart.

### 4. Units Sold by Region
Displays units sold across regions using a line chart.
